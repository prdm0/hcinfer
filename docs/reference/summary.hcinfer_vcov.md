# Summarize heteroskedasticity-consistent covariance objects

Builds a detailed summary for an object returned by
[`vcov_hc()`](https://prdm0.github.io/hcinfer/reference/vcov_hc.md).

## Usage

``` r
# S3 method for class 'hcinfer_vcov'
summary(object, ...)
```

## Arguments

- object:

  An object returned by
  [`vcov_hc()`](https://prdm0.github.io/hcinfer/reference/vcov_hc.md).

- ...:

  Unused.

## Value

An object of class `summary_hcinfer_vcov`.
