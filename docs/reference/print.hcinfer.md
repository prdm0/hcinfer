# Print hcinfer objects

Prints a compact overview of a heteroskedasticity-consistent inference
object. Emoji markers are used when the current locale supports UTF-8
and `getOption("hcinfer.use_emoji", TRUE)` is true.

## Usage

``` r
# S3 method for class 'hcinfer'
print(x, ...)
```

## Arguments

- x:

  An object returned by
  [`hcinfer()`](https://prdm0.github.io/hcinfer/reference/hcinfer.md).

- ...:

  Unused.

## Value

The input object, invisibly.
